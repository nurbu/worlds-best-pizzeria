# World's Best Pizzeria

## Description of the Project

A simple CLI based Java pizza ordering system. Intended users are customers,
who want a quick and simple way to order pizza or drink or just a garlic knot.
Users are able to order custom pizzas where they can pick everything from size to
every little details of toppings. Also, if you're indecisive you can even choose one
of our pre-designed pizzas, which you can even later customize if you chose so. Lastly,
once user's done ordering they can do a simple checkout where they will be able to get
their receipt.

## User Stories

- As a customer, I want to start a new order or exit the application from the home screen, so that I can either begin
  placing an order or close the app when I'm done.
- As a customer, I want an order screen, so I can see the different available options of what I can do with my order.
- As a customer, I want to be able to add a custom pizza to my order, so I can get the pizza perfectly the way I like
  it.
- As a customer, I want to be able to add a drink to my order, so I don't stay thirsty.
- As a customer, I want to be able to add garlic knots to my order, so I can enjoy a delicious side.
- As a customer, I want to be able to remove an item from my order, so if it's too expensive I can change my mind.
- As a customer, I want to be able to cancel the order I made, so if I don't want it I can restart.
- As a customer, I want to be able to buy the order I made, so I can enjoy my food.

## Tasks

- Create an Item interface.
- Create an abstract Topping class.
- Create RegularTopping, Sauce, and Side classes.
- Create an abstract PremiumTopping class.
- Create Meat and Cheese classes.
- Create a Pizza class.
- Create a Drink class.
- Create a GarlicKnot class.
- Create an Order class to store all the details of a customer's order.
- Create a ReceiptManager class to create a receipt for the user after checking out.
